# AssetCare24 - Структура навигации

## Обзор
AssetCare24 - платформа для управления домашним хозяйством в Германии с поддержкой ролей: Клиенты, Индивидуальные мастера, Компании-мастера, Администраторы.

## Основная навигация

### Главная страница (`/`)
- **Hero Section** - Основное предложение
- **What Is Section** - Что такое AssetCare24
- **How It Works** - Как это работает
- **Target Audience** - Целевая аудитория
- **Benefits** - Преимущества
- **Why AssetCare24** - Почему выбрать нас
- **Services Selection** - Выбор услуг
- **Getting Started** - Начало работы
- **Stats** - Статистика
- **Security** - Безопасность
- **Testimonials** - Отзывы

### Аутентификация и регистрация

#### `/anmelden`
- Форма входа для всех пользователей
- Возможность восстановления пароля

#### `/registrierung`
- Выбор роли пользователя:
  - **Kunde** (Клиент) - ищет помощь
  - **Handwerker** (Мастер) - предлагает помощь
- Перенаправление на соответствующие формы регистрации

#### `/handwerker-registrieren`
- Регистрация индивидуальных мастеров
- Форма с личными данными, специализацией, опытом

### Панели управления (Dashboard)

#### `/dashboard/admin` - Админ-панель
- Управление пользователями
- Статистика платформы
- Модерация контента
- Системные настройки

#### `/dashboard/firma` - Панель компаний
- Управление компанией
- Добавление сотрудников
- Управление заказами
- Финансовая отчетность

#### `/dashboard/handwerker` - Панель индивидуальных мастеров
- Личные заказы
- Профиль и портфолио
- Расписание работ
- Финансовая информация

#### `/dashboard/kunde` - Панель клиентов
- Создание заявок на услуги
- Отслеживание заказов
- История работ
- Управление имуществом

### Специализированные страницы

#### `/leistungen` - Услуги
- Каталог всех доступных услуг
- Поиск и фильтрация услуг
- Ценообразование

#### `/meldung` - Создание заявки
- Форма для создания новой заявки на услугу
- Выбор типа услуги, описания проблемы, предпочтений

#### `/jobs` - Вакансии
- Список открытых вакансий
- Форма подачи резюме
- Информация о компании

### Статические страницы

#### `/datenschutz` - Политика конфиденциальности
- GDPR compliant
- Информация об обработке данных

#### `/impressum` - Импринт
- Юридическая информация
- Контактные данные

#### `/debug-form` - Отладочная форма
- Тестирование форм и функционала

## Навигационные компоненты

### Header
- Логотип AssetCare24
- Основная навигация (Startseite, Leistungen, Jobs, Kontakt)
- Кнопки входа/регистрации

### Footer
- Контактная информация
- Социальные сети
- Ссылки на статические страницы

### Floating Contact
- Плавающая кнопка связи
- Быстрый доступ к WhatsApp/Telegram

## MVP упрощения

### Удаленные элементы (для упрощения MVP):
- Роль "Handwerksunternehmen" (компании-мастера)
- Все связанные страницы и формы
- Комплексная админ-панель
- Некоторые секции главной страницы (если не критичны)

### Сохраненные элементы:
- Роль "Einzelhandwerker" (индивидуальные мастера)
- Основной поток: Клиент → Заявка → Мастер
- Базовые формы регистрации и входа
- Основные услуги и функционал
