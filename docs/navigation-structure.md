# AssetCare24 - Структура навигации (MVP v1.1)

## Обзор
AssetCare24 - платформа для управления домашним хозяйством в Германии с поддержкой ролей: Клиенты, Индивидуальные мастера, Администраторы.

**Текущий статус:** MVP (Minimum Viable Product) с базовым функционалом для индивидуальных мастеров. Функции для компаний-мастеров отложены до следующих версий.

**Версия документа:** 1.1
**Дата последнего обновления:** 30 декабря 2025 г.
**Коммит:** ce8c7f6
**Репозиторий:** https://github.com/artemtech007/v0-asset-care24-1

### Аутентификация и регистрация

#### `/anmelden`
- Форма входа для всех пользователей
- Возможность восстановления пароля

#### `/registrierung`
- Выбор типа мастера для регистрации:
  - **Einzelhandwerker** (Индивидуальный мастер) - активная опция
  - **Handwerksunternehmen** (Компания-мастер) - "Demnächst" (скоро)
- Многошаговая форма регистрации индивидуальных мастеров

#### `/handwerker-registrieren`
- Альтернативная форма регистрации индивидуальных мастеров
- Упрощенная версия с основными полями

### 🔄 Доступные функции:
- **Регистрация индивидуальных мастеров** (Einzelhandwerker)
- **Панели управления** для мастеров, клиентов и администраторов
- **Базовый поток:** Клиент → Заявка → Мастер
- **Формы аутентификации** и базовые сервисы

### 🚀 Планы развития:
